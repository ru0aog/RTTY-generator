Скрипт на питоне, который генерирует и воспроизводит RTTY-сигнал в реальном времени через звуковую карту.
После записывает его же в WAV-файл (44,1 кГц 16 бит моно).
Включена поддержка кириллицы :)
Частоты по умолчанию 1000 Гц (спейс) и 1170 Гц (марк), без инверсии. Но можно задать любые, как и частоту дискретизации и скорость.
